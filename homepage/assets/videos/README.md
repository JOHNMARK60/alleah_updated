# Eventify Hero Videos

Place optimized hero background clips in this folder using these filenames:

- `eventify-wedding-reception.mp4`
- `eventify-birthday-celebration.mp4`
- `eventify-venue-decor.mp4`

Use short 8-12 second muted clips, 1080p or 720p, compressed for web playback.
The homepage will use the existing fallback image if any video is missing or cannot load.
